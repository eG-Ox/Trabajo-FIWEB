Metodologías BEM y camelCase

Página informativa sobre buenas prácticas en el nombramiento de clases CSS y variables.

¿Qué es BEM?

BEM, que significa "Block, Element, Modifier", es una metodología de desarrollo para CSS que ayuda a organizar y reutilizar estilos de manera más eficiente. Permite crear bloques de estilos (bloques), elementos dentro de esos bloques (elementos), y modificadores que cambian el estilo de los elementos. Esta metodología facilita la creación de componentes reutilizables y escalables.

Ventajas de usar BEM:

  1. Claridad al estructurar el código
  2. Mejor mantenimiento
  3. Facilita la reutilización
  4. Evita conflictos de nombres

Ejemplos de código BEM

  • .button (Bloque)
    <button class="button">Enviar</button>
  • .button__icono (Elemento)
    <button class="button"><span class="button__icono">📩</span>Enviar</button>
  • .button--activo (Modificador)
    <button class="button button--activo">Guardado</button>

¿Dónde se usa camelCase?

  1. Variables en JavaScript
    let miVariable = 10;
    Ejemplo: table (no usa camelCase, pero tablaDeDatos sí lo aplicaría).
  2. Funciones/métodos
    function obtenerDatos() { ... }
    Ejemplo: calcularTotal(), convertirAJSON().
  3. Propiedades de objetos
    const usuario = { nombreCompleto: "Ana López" };
    Ejemplo: precioConIva, esUsuarioPremium.

Comparación entre BEM y camelCase

Característica BEM camelCase
Uso principal CSS (estructuración de clases) Programación (variables y funciones)
Separación Guiones bajos dobles (`__`) y dobles guiones (`--`) Mayúsculas internas
Ejemplo .bloque__elemento--modificador miVariableEjemplo
Legibilidad Alta en estructuras CSS complejas Alta en lógica de programación
Ventaja Organización clara y mantenimiento sencillo en CSS Brevedad y coherencia en la mayoría de lenguajes
Ámbito HTML/CSS JavaScript, Java, C#, entre otros
Convención .bloque__elemento--modificador nombreDeVariable o nombreDeFuncion()
Popularidad Muy usado en desarrollo frontend moderno Estándar en múltiples lenguajes de programación

Usa BEM cuando trabajes con clases CSS. Usa camelCase cuando declares variables o funciones en lenguajes de programación.
